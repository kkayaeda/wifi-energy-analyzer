Wi-Fi Energy Analyzer is a tool that helps you detect all devices connected to your computer’s Wi-Fi and analyze their estimated energy consumption.

The tables automatically scan every 30 seconds and refresh every 10 seconds. The toggle switch allows users to switch between dark mode and light mode.

The energy chart allows for a more visual representation of the data.

It uses Python, Flask and JavaScript for the backend to scan devices and refresh the data, while HTML, CSS, and Bootstrap handle the design and layout of the website.

In the history page, you can view all energy data saved on your computer, the website also allows you to save the tables as PDF.

The full source code and project details can be found on my GitHub repository
